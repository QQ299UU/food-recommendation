# 小米应用市场上架指南

## 📋 上架前准备清单

### 1. 开发者账号 ✅
- 注册地址：https://dev.mi.com
- 完成实名认证（个人开发者需要身份证）

### 2. 应用材料准备

#### 应用图标
需要以下尺寸的图标：
- 36x36 PNG (ldpi)
- 48x48 PNG (mdpi)
- 72x72 PNG (hdpi)
- 96x96 PNG (xhdpi)
- 144x144 PNG (xxhdpi)
- 192x192 PNG (xxxhdpi)

我们的应用图标已生成在：
- `public/icon-192x192.svg` (192x192)
- `public/icon-512x512.svg` (512x512)

#### 应用截图
需要 3-5 张应用截图，尺寸：1080x1920 像素

#### 应用描述
简短描述（50字以内）：
```
食探地图是一款专注于美食推荐与探索的应用，帮您发现身边的美味餐厅，分享真实用餐体验。
```

详细介绍：
```
【发现美食】
- 探索附近的热门餐厅和小吃
- 根据口味偏好智能推荐
- 查看其他用户的真实评价

【分享体验】
- 发布美食笔记和点评
- 上传美食照片
- 与好友分享美味发现

【商家入驻】
- 商家可申请入驻平台
- 展示店铺信息和优惠活动
- 管理店铺评价和排名

特色功能：
- 精准定位周边美食
- 真实用户点评
- 丰富的美食资讯
```

### 3. 隐私政策

必须提供隐私政策网页，可以先创建一个简单的页面放在网上。

## 📦 安装包说明

当前应用已部署到 Vercel，可以通过以下方式安装：

### 方式一：PWA 安装（推荐）
1. 用手机浏览器访问：`https://ffud-recomkqendation.vercel.app`
2. 点击浏览器菜单，选择"添加至主屏幕"
3. 应用会像原生APP一样出现在桌面

### 方式二：生成 APK
需要将 Web 应用打包成 Android 安装包

## 🚀 上架流程

### 1. 登录小米开放平台
访问：https://dev.mi.com

### 2. 创建应用
- 应用名称：食探地图
- 应用类型：生活娱乐 → 餐饮美食
- 包名：com.example.foodrecommendation

### 3. 上传应用材料
- 应用图标
- 应用截图
- 应用描述
- 隐私政策链接

### 4. 上传安装包
- APK 文件（需签名）

### 5. 提交审核
- 审核时间：1-3 个工作日
- 审核通过后自动上架

## ⚠️ 注意事项

1. 个人开发者可以上架免费应用
2. 应用内不能有违规内容
3. 商家入驻功能需确保商家资质审核
4. 需要提供有效的隐私政策链接

## 📞 遇到问题？

小米开发者平台有详细的帮助文档和客服支持。
