# 食探地图 - 应用安装说明

## 📱 当前安装方式

### 方法一：手机浏览器安装（推荐）

1. **用手机浏览器打开**：
   - 访问 `https://ffud-recomkqendation.vercel.app`
   
2. **添加到主屏幕**：
   - iPhone：点击分享按钮 → "添加到主屏幕"
   - Android：点击右上角菜单 → "添加到主屏幕" 或 "安装应用"

3. **安装完成**：
   - 应用会像原生APP一样出现在手机桌面
   - 可以离线使用（部分功能）

### 方法二：桌面安装包

已生成的 Windows 安装包位置：
- `dist/electron/食探地图 Setup 1.0.0.exe`

## ⚠️ 关于上架应用商店

要上架到小米应用市场，需要将 Web 应用转换为原生 Android 应用（APK）。这需要：

1. **使用打包工具**：如 Capacitor、Tauri 等
2. **生成签名**：为 APK 文件签名
3. **准备材料**：
   - 应用图标（多种尺寸）
   - 应用截图
   - 隐私政策页面
   - 应用描述文案

## 📋 准备上架所需材料

### 1. 应用图标
需要准备以下尺寸的 PNG 图标：
- 36×36、48×48、72×72、96×96、144×144、192×192 像素

我们的图标文件：
- `public/icon-192x192.svg`
- `public/icon-512x512.svg`

### 2. 应用截图
需要 3-5 张手机截图，尺寸：1080×1920 像素

### 3. 隐私政策页面
需要一个可访问的隐私政策网页，可以放在 Vercel 上。

### 4. 应用描述
简短描述：食探地图是一款专注于美食推荐与探索的应用，帮您发现身边的美味餐厅，分享真实用餐体验。
